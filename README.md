# உயர் வள்ளுவம்

இறையருளால் இலங்கை ஜெயராஜ் ஐயா பரிமேலழகரின் உரை அடிப்படையில் நடத்தும் திருக்குறள் தொடர் வகுப்புகள், உயர் வள்ளுவம். 
இது, திருக்குறள் முலமாக அறத்தை எல்லோரின் வாழ்வியலாக்கும் முயற்சி.
